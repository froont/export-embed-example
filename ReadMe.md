## Froont export embed example:
This is example of exported code from Froont app, to be used as embed in another page.

## Base Style and Script
The froont-base-script.js and froont-base-style.css are Froont global CSS and script files. They are provided here for user to be able to store them in their own CDN (the paths to files should be updated in HTML).
The froont-base-script.js and froont-base-style.css can also be provided embedded in the HTML.
